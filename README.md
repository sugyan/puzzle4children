# ちいくパズル メーカー

小学校低学年（6歳前後）向けの「法則・パターン発見」型 知育パズルを自動生成し、
A4 に印刷して鉛筆で解くための静的サイト。Vite + TypeScript、描画は手書き SVG のみ。

## つかいかた

```bash
npm install
npm run dev      # 開発サーバ（http://localhost:5173）
```

画面で「しゅるい / むずかしさ / まいすう / シード」を選び「もんだいをつくる」。
「いんさつ」→ ブラウザの印刷ダイアログで **用紙 A4 / 余白なし（最小）** にして印刷（PDF化可）。
問題ページ群のあとに答えページ群が並び、1枚ずつ改ページされる。シードを隅に印字するので再現可能。

```bash
npm run build    # 型チェック + 本番ビルド（dist/）
npm test         # 生成ロジックの検証（経路の正当性・全面タイリング・再現性）
```

## パズル

- **きまりめいろ**: グリッド各マスに記号。スタート→ゴールを、上部に示した周期パターン順に辿る。
  自己回避ウォークに `cycle[i % k]` を置いて正解パスを作る（唯一解は不要）。
- **しきつめ**: 枠をポリオミノで全面タイリング（バックトラック）し、その分割を答えとする。
  出題は空の枠＋つかうブロック一覧。

記号は白黒印刷でも形で区別できる自作 SVG アイコン（`src/icons.ts`）。
答えのハイライトは色＋番号・太線で、白黒でも判別できる。

## パズルを増やす

`src/types.ts` の `PuzzleType` を実装し、`src/puzzles/registry.ts` の配列に1行足すだけ。

```ts
export const puzzles: PuzzleType[] = [ruleMaze, tiling /*, newPuzzle */];
```
